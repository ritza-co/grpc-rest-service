# SDK


## Overview

Hitchhiker's Guide to Drinks: An API for creating and retrieving drinks.

### Available Operations

* [createDrink](#createdrink) - Create a new drink based on provided ingredients
* [getDrink](#getdrink) - Get a drink recipe by providing its name

## createDrink

Create a new drink based on provided ingredients

### Example Usage

```typescript
import { SDK } from "openapi";

async function run() {
  const sdk = new SDK();

  const result = await sdk.createDrink({
    ingredients: [
      "Gin",
      "Tonic water",
      "Lime juice",
    ],
  });

  // Handle the result
  console.log(result)
}

run();
```
